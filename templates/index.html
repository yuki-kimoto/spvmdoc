SPVM Document

SPVM transpiles Perl program to C program with minor modifications and runs it with the fast performance of C language "-O3" optimization. SPVM has utilities for running native languages such as C/C++/Cuda for even faster peformance and parallelization. Try the SPVM benchmark once. The result is amazing.
